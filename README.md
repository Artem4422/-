# Telegram Парсер

Приложение для мониторинга чатов Telegram по ключевым словам и рассылки сообщений.

## Запуск

### Windows
```
install.bat
run.bat
```

### macOS
См. `install_mac.txt`

## Сборка

- Windows: `build_win.bat`
- macOS: `./build_mac.sh`

## Настройка

Заполните `config.json`:
- `api_id` и `api_hash` — https://my.telegram.org
- `phone` — номер телефона в формате `+7...`
